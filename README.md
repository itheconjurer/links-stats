# links-stats
Accept some links from user, and find their stats such as:
- Status code of response
- content type of response from link 
- tags and their occurance counts
- if response is JSON, key names at the root of JSON

This project demonstrates the following:

- Django project
- Asynchronous views
- Class based views
- Async aware Django ORM queries
- Login protection using async mixins
- Accept link from user, avoid duplicates
- Async parallel execution to download content at link
- HTML Parsing
- Async HTTP Client using AIOHTTP
- Simple UI using pure HTML Forms
